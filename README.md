# Linux-Update-Notifier


Simple linux package update notification script

#### Usage

Clone the repo and install with the install script

```bash
git clone https://github.com/Spcktr/Linux-Update-Notifier

cd Linux-Update-Notifier
chmod +x install.sh
 ./install.sh
```
